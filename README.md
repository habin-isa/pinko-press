# Pinko Press

### Static site for Pinko Press publishing house (Tommy Spitters)

### To run the project

- \$ git clone https://github.com/habin-isa/pinko-press.git
- \$ cd pinko-press
- \$ yarn
- \$ yarn start

- \$ yarn test

### Examples contained

- Styled components
- Jest testing
- Proptypes

### Tech

- React CLI
- Pretty-quick
- Styled components
- Jest
- Axios
- Babel/EsLint
